## 🚀 Deployment Instructions

### Quick Deploy (15 minutes)

1. **Push to GitHub**
   ```bash
   git push origin master --tags
   ```

2. **Deploy on Streamlit Cloud**
   - Go to share.streamlit.io
   - Click "New app"
   - Select repository
   - Set main file: app.py
   - Click "Deploy"

3. **Verify Deployment**
   - Test all pages
   - Check functionality
   - Update README with URL

### Full Setup (with models)

1. **Train Models** (2-3 hours per disease)
   - Open notebooks in Google Colab
   - Run training pipeline
   - Download .pkl files

2. **Upload Models**
   - Place .pkl files in models/
   - Commit and push

3. **Test End-to-End**
   - Upload sample CSVs
   - Verify predictions
   - Check SHAP visualizations
